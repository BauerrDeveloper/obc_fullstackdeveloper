Procurando Palíndromo

Escreva um programa em JS que seja capaz de identificar se uma palavra é um palíndromo.

Um palíndromo é uma palavra que lida de trás para frente possui as mesmas letras na mesma ordem.

O programa deve iniciar pedindo que seja informada uma palavra e então deve exibir uma mensagem dizendo se a palavra é um palíndromo ou não.

Caso seja um palíndromo, o programa deve mostrar a palavra lida da esquerda para a direita e da direita para a esquerda.
