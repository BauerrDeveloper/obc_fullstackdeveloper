Formulários

Dentro da tag form, temos atributo action que é onde fica a url pra onde o form vai ser enviado

Tag label é um texto associado com um input. Quando clica já seleciona, isso é feito com o atributo for para um campo que tem o mesmo id que escolhemos no atributo for.

Input vai ter o id, que é o mesmo id do campo for da tag label. Assim, o label ficará associado com o input.
No atributo name, também passamos o mesmo que em for e em id.

Input do tipo button cria o botão.

Validação de forms

Atributo required faz com que o campo seja de preenchimento obrigatório. Colocado no campo desejado.

Validação de e-mail: tipo do campo deve ser e-mail

Validação min e máx. Tipo do campo number passando atributo min="10 max="12"
